# EncuestasSIAU

Aplicación Android desarrollada en Kotlin + Jetpack Compose para la recolección
de encuestas de satisfacción en el Hospital Universitario San José de Popayán.

## Tecnologías
- Kotlin
- Jetpack Compose
- Room
- Ktor
- Coroutines

## Características
- Funciona offline
- Sincronización automática
- Exportación CSV
